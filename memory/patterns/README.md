# Candidatos para Destilação de Skills

Features concluídas com `attempts_coder: 1` e tester aprovado na primeira tentativa. São candidatos para refinamento de skills existentes ou criação de novos.

## Ciclo de revisão
1. Revisar `candidates.md` semanalmente
2. Para cada entrada, abrir o `specs.md` da feature
3. Identificar padrão reutilizável → atualizar skill ou criar novo com `/start nome-skill`
4. Marcar como revisado com `[x]` na entrada
